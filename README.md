# Práctica 1: Operaciones Básicas sobre Lenguajes con Interfaz Gráfica

## Descripción
Este proyecto implementa una aplicación con interfaz gráfica desarrollada en Python utilizando la librería *Flet*.  
El programa permite realizar operaciones básicas sobre lenguajes formales, como:

- Cálculo de *prefijos*
- Cálculo de *sufijos*
- Generación de *subcadenas*
- Generación de *cerradura de Kleene (Σ\)**
- Generación de *cerradura positiva (Σ+)*

Además, el usuario puede guardar los resultados en un archivo de texto.

---

## Requisitos

Antes de ejecutar el programa es necesario tener instalado:

- Python 3.14.3
- Librería *Flet*

---

## Instalación

Primero se debe instalar la librería Flet con el siguiente comando:


pip install flet

## Cómo ejecutar el programa

1. Descargar o clonar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar el siguiente comando:


python practica_lenguajes.py


4. Se abrirá una ventana con la interfaz gráfica del programa.

---

## Uso del programa

### Subcadenas, Prefijos y Sufijos
1. Ingresar una cadena en el campo correspondiente.
2. Presionar el botón *Calcular*.
3. El programa mostrará los prefijos, sufijos y subcadenas generadas.

### Cerradura de Kleene y Positiva
1. Ingresar el alfabeto separado por comas (ejemplo: 0,1).
2. Ingresar la longitud máxima de las cadenas.
3. Presionar *Generar Lenguaje*.

---

## Guardar resultados

El programa incluye un botón para guardar los resultados en un archivo:


resultado.txt


Este archivo se guarda en la misma carpeta donde se encuentra el programa.
